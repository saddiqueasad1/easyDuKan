## easydukan

### Overview

**easydukan** is a mobile application built using React Native and Expo. It helps users manage their daily diet plans and provides easy access to nutrition information.

### Features

- **Diet Management**: Users can create and customize their daily diet plans.
- **Nutrition Information**: Access detailed nutrition information for various foods.
- **Cross-Platform Compatibility**: The app is built using React Native and Expo, ensuring compatibility with both iOS and Android platforms.
- **User-Friendly Interface**: Intuitive design and easy navigation make it simple for users to track their diet.

### Installation

To run the app locally, follow these steps:

1. Make sure you have Node.js and npm installed on your machine.
2. Clone this repository.
3. Navigate to the project directory in your terminal.
4. Run `yarn install` to install dependencies.
5. Start the Expo development server by running `yarn start`.
6. Use the Expo Go app on your mobile device to scan the QR code generated by the Expo development server.

### Tech Stack

- **React Native**: A JavaScript framework for building native mobile applications.
- **Expo**: A set of tools and services for building and deploying React Native apps.
- **TypeScript**: A statically typed superset of JavaScript that enhances code quality and developer productivity.
- **ESLint & Prettier**: Tools for maintaining code quality and enforcing code style consistency.

### Scripts

- `yarn start`: Start the Expo development server.
- `yarn android`: Start the app on an Android device or emulator.
- `yarn ios`: Start the app on an iOS simulator.
- ` npx expo lint`: Run ESLint to check for code style and formatting issues.
